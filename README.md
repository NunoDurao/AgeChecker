# AgeChecker 

## Description

This is a simple age checker web application developed using HTML, CSS, and JavaScript. The program prompts the user to enter their gender and year of birth. Based on this information, it calculates the user's age and displays a corresponding photo, categorizing them as a child, young person, adult, or senior citizen.

## Features

- User-friendly interface.
- Age calculation based on the entered year of birth.
- Display of a corresponding photo based on the user's age category.

## Technologies Used

- HTML
- CSS
- JavaScript

## Usage

1. Open the `index.html` file in a web browser.
2. Enter your gender and year of birth as prompted.
3. The program will calculate your age and display a corresponding photo.
